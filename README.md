## ⑤工夫した点・こだわった点

- 「乱数」はデータ生成のためにまずは、モデルを作成しました。
【具体的には以下の通り】
　・来店客数の分布：平均350名　標準偏差50人の正規分布をしている
　・客単価の分布：平均750円　標準偏差300円の正規分布をしている
ただし、両方の値を共に、整数としてマイナスの値は除外する。
当該モデルにのっとった乱数をそれぞれ10000回発生させて、各数字を生成しそこから売上高を計算した。
それらの計算結果をSQL上に格納し、データをPHP等で出力した。
